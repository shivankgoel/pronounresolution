WSJ.pron contains BBN's pronoun annotation of the WSJ corpus. 
Pronouns and antecedents are listed by sentence and token numbers. 

So...

(WSJ1015
    (
	Antecedent -> S34:11-12 -> Ford Motor
	Pronoun -> S34:17-17 -> it
    )
)

specifies that in WSJ1015 there is a pronoun "it" that is the 17th token 
(one-indexed) of the 34th sentence. The "it" has the antecedent "Ford 
Motor" which is the 11th and 12th token of the 34th sentence.

WSJ.sent lists all the WSJ sentences.
